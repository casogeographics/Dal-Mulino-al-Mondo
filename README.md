# Dal Mulino al Mondo — De Cecco Legacy Game

Concept demo browser-based, non ufficiale, realizzato come serious game interattivo.

## Pubblicazione
Caricare nella root della repository:
- `index.html`
- cartella `assets/`

Attivare GitHub Pages su `main / root`.

## Contenuti
Il progetto conserva le quattro epoche e le dodici informazioni storiche presenti nell'index originale, trasformandole in missioni interattive: mappa territoriale, simulazioni di processo, allocazione di risorse, ricostruzione della linea, portfolio, rotte globali, archivio e atlante delle ricette.

## Visual identity
This edition uses the same core palette and typography as the original supplied game: De Cecco blue `#00529b`, dark blue `#003a70`, yellow `#f5b91e`, cream `#fbf4e4`, red `#c8102e`, Fraunces for display titles and Archivo for interface text. Gameplay and content are unchanged.
